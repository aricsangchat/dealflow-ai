# DealFlow AI

AI-powered real estate investment analyzer for ranking property deals based on projected cash flow, cap rate, cash-on-cash return, and neighborhood fundamentals.

## Tech Stack

- Next.js
- React
- TypeScript
- Tailwind CSS

## Features

- Market-based property dashboard
- Investment score ranking
- Cash flow and ROI calculations
- AI-style investment summaries
- Property detail pages

## Goal

This project is being built as a modern SaaS-style MVP to demonstrate product thinking, financial modeling, and frontend engineering.

## Status

MVP in progress.
